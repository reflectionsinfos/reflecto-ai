Executive Summary: This release introduces several improvements to the deployment process, release notes generation, and various bug fixes and infrastructure updates for the reflecto-ai service. These changes aim to enhance the overall stability and efficiency of the system. The updates include a new release notes viewer component, improved error handling, and enhancements to the deployment process. Additionally, the dashboard layout has been updated to display release numbers and include a release notes dialog.

⚠ Breaking Changes

  • Changes to error handling and deployment process (reflecto-ai) may require updates to dependent systems

  • Added deployment log, release notes generation, and remote server layout documentation (reflecto-ai)
  • Introduced a release notes viewer component (reflecto-ai)
  • Updated dashboard layout to display release number and include a release notes dialog (reflecto-ai)
